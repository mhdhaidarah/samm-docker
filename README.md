# SAMM — Docker Compose distribution

[SAMM](https://samm.securytik.com) (SecuryTik Active MikroTik Manager) packaged
as Docker Compose for ISP on-prem installs.

This repository is **releases-only** — its source is generated by CI from the
private [`mhdhaidarah/samm-code`](https://github.com/mhdhaidarah/samm-code)
repo on each version-tag push. Every release carries:

- `install.sh` — one-shot bootstrap (installs Docker, drops compose, brings up)
- `docker-compose.yml` — pinned to the matching `mhdhaidarah/samm@sha256:…` image
- `host-updater.sh` — cron-friendly auto-upgrade script
- `.env.example` — required env vars
- `README.md` — quickstart, network requirements, v1 limitations

## Quick install (Ubuntu / Debian)

```bash
curl -fsSL https://github.com/mhdhaidarah/samm-docker/releases/latest/download/install.sh \
    | sudo bash
```

The installer detects Docker, installs it if missing, drops a
`docker-compose.yml` + `.env` into `/opt/samm-docker/`, and brings the stack up.

After ~30 seconds:

- **Admin portal**     `http://<host>:8000/admin`
- **Customer portal**  `http://<host>:8000/`
- **RADIUS auth**      UDP 1812 on the host's LAN IP
- **RADIUS accounting** UDP 1813 on the host's LAN IP

Default credentials: `admin` / `samm` — **change on first login**.

## Bare-OS alternative

If you'd rather not run Docker, the bare-OS install lives at
[`mhdhaidarah/samm`](https://github.com/mhdhaidarah/samm). Same product,
different packaging — pick whichever fits your host.
